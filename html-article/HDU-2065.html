HDU-2065 "红色病毒"问题

题目链接:http://acm.hdu.edu.cn/showproblem.php?pid=2065

题目大意

有A,B,C,D四种字符,其中字符A,C只能出现偶数次,其他字符没有要求.给出一个 N N N要求按照上述规则构造一个字符串,求方案数 ( m o d 100 ) \pmod{100} (mod100),其中 1 ≤ N ≤ 2 64 1 \leq N \leq 2^{64} 1≤N≤264.

解析

使用指数型生成函数:

从题目不难看出:
A ^ ( x ) = C ^ ( x ) = 1 + 1 2 ! x 2 + 1 4 ! x 4 … B ^ ( x ) = D ^ ( x ) = 1 + x + 1 2 ! x 2 … \begin{aligned} &\hat{A}(x)=\hat{C}(x)=1+\dfrac{1}{2!}x^{2}+\dfrac{1}{4!}x^{4}\dots \\ &\hat{B}(x)=\hat{D}(x)=1+x+\dfrac{1}{2!}x^{2}\dots \\ \end{aligned} ​A^(x)=C^(x)=1+2!1​x2+4!1​x4…B^(x)=D^(x)=1+x+2!1​x2…​
所以总体的生成函数为:
G ^ ( x ) = A ^ ( x ) × B ^ ( x ) × C ^ ( x ) × D ^ ( x ) = ( 1 + 1 2 ! x 2 + 1 4 ! x 4 …  ) 2 × ( 1 + x + 1 2 ! x 2 …  ) 2 \begin{aligned} \hat{G}(x) &=\hat{A}(x)\times\hat{B}(x)\times\hat{C}(x)\times\hat{D}(x) \\ &=(1+\dfrac{1}{2!}x^{2}+\dfrac{1}{4!}x^{4}\dots)^{2}\times(1+x+\dfrac{1}{2!}x^{2}\dots)^{2} \end{aligned} G^(x)​=A^(x)×B^(x)×C^(x)×D^(x)=(1+2!1​x2+4!1​x4…)2×(1+x+2!1​x2…)2​
根据泰勒展开:
e x = 1 + x + 1 2 ! x 2 … e − x = 1 − x + 1 2 ! x 2 … \begin{aligned} e^{x}&=1+x+\dfrac{1}{2!}x^{2}\dots \\ e^{-x}&=1-x+\dfrac{1}{2!}x^{2}\dots \end{aligned} exe−x​=1+x+2!1​x2…=1−x+2!1​x2…​
所以:
G ^ ( x ) = ( e x + e − x 2 ) 2 × ( e x ) 2 = e 4 x + 2 e 2 x + 1 4 \begin{aligned} \hat{G}(x) &=(\dfrac{e^{x}+e^{-x}}{2})^{2}\times(e^{x})^{2} \\ &=\dfrac{e^{4x}+2e^{2x}+1}{4} \end{aligned} G^(x)​=(2ex+e−x​)2×(ex)2=4e4x+2e2x+1​​
又因为:
e 2 x = 1 + 2 x + 2 2 2 ! x 2 … e 4 x = 1 + 4 x + 4 2 2 ! x 2 ⋯ \begin{aligned} e^{2x}&=1+2x+\dfrac{2^{2}}{2!}x^{2}\dots \\ e^{4x}&=1+4x+\dfrac{4^{2}}{2!}x^{2}\dotsb \end{aligned} e2xe4x​=1+2x+2!22​x2…=1+4x+2!42​x2⋯​
所以答案就是 x n x^{n} xn的系数乘以 n ! n! n!,即 2 n − 1 + 4 n − 1 ( m o d 100 ) 2^{n-1}+4^{n-1} \pmod{100} 2n−1+4n−1(mod100)
